Desafío 1 · Veterinaria JS CLI

Gestor de citas en Node.js que usa File System para registrar, listar y consultar detalles desde la línea de comandos.

Prerrequisitos

  • Node.js 18+
  • Consola con acceso al proyecto
  • JSON `citas.json` en la raíz (se crea si no existe)

Registrar cita

node index.js registrar <nombre> <edad> <animal> <color> <enfermedad>
  • Genera un id corto y fecha ISO.
  • Edad se normaliza a número si es posible.

Listar citas

node index.js listar
  • Muestra todas las citas almacenadas.
  • Si alguna cita antigua no tiene id, se etiqueta como "sin-id".

Detalle por id

node index.js detalle <id>
  • Imprime el objeto completo en JSON.
  • Útil para revisar campos guardados.
